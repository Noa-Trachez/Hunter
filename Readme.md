# My Hunter
Mon premier jeux vidéo en C !

## Projet

Ce jeu a été réalisé en C avec la librairie CSFML lors de ma première année d'étude.

Il est inspiré du Duck Hunt sorti sur NES en 1987.

## Jouer au jeu

### Prérequis
    - CSFML
    - C
    - Make

### Lancer le projet
```bash
make
./my_hunter
```
